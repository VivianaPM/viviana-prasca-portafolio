# Portafolio Profesional

Este repositorio contiene un portafolio profesional desarrollado con MkDocs y Material para MkDocs. Su propósito es presentar proyectos, experiencias y recursos de forma clara y visualmente atractiva.

## 🚀 Tecnologías usadas

- Python
- MkDocs
- Material for MkDocs

## 📂 Estructura del proyecto

```text
.
├── docs/
│   ├── index.md
│   ├── assets/
│   │   └── images/
│   ├── proyecto-demeter/
│   ├── proyecto-hubentory/
│   └── proyecto-QualityClassifier/
├── mkdocs.yml
└── README.md
```

## ▶️ Cómo ejecutar el sitio localmente

1. Instala las dependencias:

```bash
pip install mkdocs mkdocs-material
```

2. Inicia el servidor local:

```bash
mkdocs serve
```

3. Abre la siguiente URL en tu navegador:

```text
http://127.0.0.1:8000/
```

## 📝 Contenido

El portafolio incluye páginas de presentación para distintos proyectos, con información técnica, contexto del problema y enlaces relevantes.
